# TFG
Proyecto de TFG para Grado Superior Desarrollo de Aplicaciones Web

## Iniciar backend
Las instrucciones para iniciar el backend del proyecto se encuentran en el `README` dentro de la carpeta `backend`.

## Iniciar frontend

### Prerrequisitos
Descargar las dependencias desde el directorio `TFG\cinesnap`:

```
npm install
```

Una vez descargadas, iniciamos el proyecto desde el mismo directorio `cinesnap`:

```
npm run dev
```
